## Scripts

To install dependencies:

```sh
yarn install
```

Start development mode:

```sh
yarn run start
```

Create production build:

```sh
yarn run build
```

Lint scripts:

```sh
yarn run lint:scripts
```

Lint styles:

```sh
yarn run lint:styles
```

Format code with Prettier:

```sh
yarn run lint:format
```
